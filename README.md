# Barekey docs

Mintlify documentation source for Barekey at [docs.barekey.dev](https://docs.barekey.dev).

## Local development

```bash
npm install -g mint
cd docs
mint dev
```

The local preview runs against [`docs/docs.json`](/home/sander/barekey/docs/docs.json).

## What lives here

- Product guides and quickstart content
- CLI, SDK, and HTTP API documentation
- Shared assets, branding, and Mintlify config

## Publishing

Docs changes are published through Mintlify after they land on the default branch for the connected repository.
