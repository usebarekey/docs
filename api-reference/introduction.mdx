## Base URL

Production API requests use:

```text
https://api.barekey.dev
```

## Current endpoint groups

### Variable evaluation

- `POST /v1/env/evaluate`
- `POST /v1/env/evaluate-batch`
- `POST /v1/env/pull`

### Variable management

- `POST /v1/env/list`
- `POST /v1/env/write`

### CLI auth

- `POST /v1/cli/device/start`
- `POST /v1/cli/device/complete`
- `POST /v1/cli/device/poll`
- `POST /v1/cli/token/refresh`
- `POST /v1/cli/logout`
- `GET /v1/cli/session`

### Tooling

- `GET /v1/typegen/manifest`

## Response shape

Successful responses are JSON.
Errors return this shape:

```json
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "A valid Clerk JWT or CLI access token is required.",
    "requestId": "..."
  }
}
```
