## What Barekey gives you

Barekey manages runtime configuration for developer teams.

Today, that means:

- encrypted variables that replace scattered `.env` files
- per-project envelope encryption
- optional `ab_roll` values when you need runtime variation
- one HTTP API used by the app, the CLI, and the JavaScript SDK
- Clerk-backed auth for browser and CLI flows

## Current product surface

Use these docs for the features that are available now:

- `secret` variables for the main `.env` replacement workflow
- `ab_roll` variables as an optional extension
- CLI device login
- variable evaluation, listing, writing, and pulling
- JavaScript SDK reads
- manifest generation for type tooling
